# <a name="msv2-and-mdsv2-series-medium-memory-preview"></a>Средняя память серии Msv2 и Mdsv2 (Предварительная версия)


> [!IMPORTANT]
> Присоединитесь к предварительной версии, заполнив форму в **https://aka.ms/Mv2MedMemoryPreview** .  

Серия виртуальных машин Msv2 и Mdsv2 с средней памятью — это процессор Intel® Xeon® Platinum 8280 (Cascade Lake) со всеми базовыми частотами частотой одноядерных операций Turbo с частотой 2,7 ГГц и 4,0 ГГц. Благодаря этим виртуальным машинам клиенты получают повышенную гибкость благодаря параметрам "локальный диск" и "без диска". У клиентов также есть доступ к набору новых изолированных виртуальных машин с большим объемом ресурсов ЦП и памяти, до 192 виртуальных ЦП с 4 Тиб памяти. 


Виртуальные машины серии Msv2 и Mdsv2 поддерживают только поколение 2 и поддерживают подмножество поддерживаемых образов версии 2. Полный список поддерживаемых образов для серии Msv2 и Mdsv2 см. ниже.  

- Windows Server 2019 или более поздней версии;
- SUSE Linux Enterprise Server 12 SP4 и более поздней версии или SUSE Linux Enterprise Server 15 SP1 и более поздних версий
- Red Hat Enterprise Linux 7,6, 7,7, 8,1 или более поздней версии 
- Oracle Enterprise Linux 7,7 или более поздней версии

Дополнительные сведения о виртуальных машинах версии 2 см. [в статье поддержка виртуальных машин поколения 2 в Azure](./generation-2.md).
