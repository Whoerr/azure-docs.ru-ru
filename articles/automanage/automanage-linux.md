# <a name="azure-automanage-for-virtual-machines-best-practices---linux"></a>Рекомендации по управлению Azure для виртуальных машин — Linux

Эти службы Azure автоматически подключаются при использовании автоматического управления для виртуальных машин на виртуальной машине Linux. Они необходимы для нашего технического документа, который можно найти в нашей [облачной инфраструктуре внедрения](/azure/cloud-adoption-framework/manage/azure-server-management).

Для всех этих служб мы будем автопереходить на автоматическую настройку, отслеживать отклонения и устранять их при обнаружении отклонения. Дополнительные сведения об этом процессе см. в разделе [Управление Azure для виртуальных машин](automanage-virtual-machines.md).

## <a name="supported-linux-distributions-and-versions"></a>Поддерживаемые дистрибутивы и версии Linux

Управление поддерживает следующие дистрибутивы и версии Linux:

- CentOS 7.3 +
- RHEL 7.4-7,8
- Ubuntu 16.04 и 18.04.
- SLES 12 (все пакеты обновления)
